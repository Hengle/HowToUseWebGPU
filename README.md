# webgpu 上手记录

webgpu 是下一代的 web 端图形接口，虽然目前仍然没法发布正式版。但是已经有可以运行的环境，可以让开发者尝鲜体验，本仓库就是记录笔者学习使用 webgpu 的整个过程。其中可能会包括如下的一些内容：

1. 最简单的应用从清屏开始
2. 画个三角形
3. 从平面到立体
